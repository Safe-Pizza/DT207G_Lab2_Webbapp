# Laboration 2 Uppgift 2, DT207G Webbtjänster
Detta är en laboration i _Webbutveckingsprogrammet_ på Mittuniveristetet.

## Syfte
Syftet med denna laboration är att bekanta dig med med begreppet webbtjänster och kunna använda Fetch API för att konsumera webbtjänsten för arbetserfarnheter skapad i uppgift 1.

## Uppgift
Du ska skapa en webbapplikation som använder FetchAPI. Applikationen ska konsumera webbtjänsten från uppgift 1 och bestå av nedan undersidor:

* **index.html:** Alla arbetserfarenheter från webbtjänst visas,
* **add.html:** Formulär för att lägga till nya poster i webbtjänst,
* **about.html:** Beskriver webbplats, dess syfte, databasinfo och dragna slutsatser.

## Lösning
Här kan du läsa om vilka funktioner som finns, vad som används samt hur repot kan installeras lokalt i din dator.

⚠️ OBS! Även repot för webbtjänsten måste installeras och köras för att detta repo ska fungera: https://github.com/safe-pizza/DT207G_Laboration2.git

### Funktioner
* **Ta bort jobb:** Jobb kan tas bort med knapp på startsidan,
* **Ändra jobb:** Jobb kan ändras med knapp på startsidan,
* **Lägga till jobb:** Uppgifter kan läggas till via formulär på sidan: "Lägg till",
* **Validering:** Inputfälten valideras, fälten får ej vara tomma och datum måste ha ett specifikt format.

### Har används för uppgiften
* HTML
* CSS
* JavaScript
* Vite

### Installation
1. Klona repot
    ```sh
    git clone https://github.com/safe-pizza/DT207G_Laboration2_Webbapp.git
    ```

2. Gå in i projektmappen
    ```sh
    cd DT207G_Laboration2
    ```

3. Installera
    ```sh
    npm install
    ```

4. Starta en utvecklingsserver
    ```sh
    npm run dev
    ```

Nu finns projektet tillgängligt för dig att köra lokalt på din dator via localhost.
